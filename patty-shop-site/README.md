# The Patty Shop Website

Static multi-page website for The Patty Shop.

## Files

- `index.html` — homepage
- `menu.html` — menu highlights page
- `about.html` — about page
- `visit.html` — visit/location page
- `contact.html` — contact page
- `assets/styles.css` — small custom styles
- `assets/main.js` — mobile menu and FAQ interactions

## Run locally

Open `index.html` in a browser.

## Deploy to GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select your main branch and the root folder.
6. Save.

Your site will publish as a static website.

## Notes

- Tailwind CSS is loaded through the CDN.
- Font Awesome is loaded through the CDN.
- Images are referenced from the provided external URLs.
- The Google Maps embed uses a standard public embed query URL.
